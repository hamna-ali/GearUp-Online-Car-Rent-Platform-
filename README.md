# 🚗 GearUp – Online Car Rent Platform

**GearUp** is a dynamic car rental web application built with ASP.NET MVC. It enables customers to book vehicles by city and category, with optional services like drivers, car washes, and event decorations. Admins can manage inventory, prices, and bookings through a dedicated dashboard.

## ✨ Features

### 👥 Role-Based System
- **Customers**: Sign up, log in, browse cars, add services, book cars, and receive invoices.
- **Admins**: Manage vehicles, bookings, services, and view all payments.

### 🚘 Car Booking Workflow
- Search cars by **category** (Luxury, Economy, Coasters, etc.)
- Filter by **city** (Lahore, Karachi, Islamabad)
- Select rental options:
  - With/Without Driver
  - Inside/Outside City
  - Pickup or Home Delivery
- Add to cart → Checkout → Download Invoice

### 🛠 Admin Panel
- Add/Edit/Delete car listings
- Mark cars as **Booked** or **Returned**
- Update extra service charges
- View booking/payment records

### 📄 Invoice Generation
- Auto-generated invoices include car rent, add-on services, and delivery costs.

## 🧰 Tech Stack

- **Framework**: ASP.NET MVC (C#)
- **Frontend**: Razor Views, Bootstrap, jQuery
- **Database**: SQL Server
- **ORM**: Entity Framework
- **Authentication**: ASP.NET Identity
